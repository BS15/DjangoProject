# Wiki Técnica do Backoffice

Esta wiki documenta a base de código do projeto Django com foco em:

- arquitetura por domínio;
- referência de módulos Python;
- fluxo de pré-pagamento, pagamento e pós-pagamento;
- utilitários e serviços de integração.

## Estrutura

Use o menu lateral para navegar pelas seções de referência técnica.

## Cobertura de workflows

A seção Workflows Operacionais documenta os fluxos completos de:

- pagamentos end-to-end;
- fiscal (liquidação e retenções);
- verbas indenizatórias;
- suprimentos de fundos;
- contingências e devoluções;
- conselho e arquivamento;
- assinaturas digitais;
- contas fixas;
- integrações e importações.

## Observações

- A documentação de API é gerada com `mkdocstrings` diretamente dos módulos Python.
- Esta wiki foi organizada para refletir o domínio de execução financeira pública.
