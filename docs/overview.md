# Visão Geral

Este sistema modela a execução orçamentária e administrativa de acordo com as normas da administração pública brasileira.

- Processos financeiros e documentais
- Controle de verbas indenizatórias, suprimentos de fundos, pagamentos
- Integrações: SISCAC, Autentique

Consulte a referência de API para detalhes técnicos.
